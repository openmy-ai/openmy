# 用户故事模板

这份模板是给你写第一篇真实案例用的。

## 标题

一句话说清楚：OpenMy 帮你省掉了什么。

示例：
- 我怎么用 OpenMy 把每天碎掉的录音收成可追踪的工作上下文
- 我怎么让 Agent 直接读我最近几天在做什么

## 开头

先写一个真实场景：

- 当时你为什么觉得痛苦
- 你原来怎么做
- 原来的办法为什么不够用

## 中段

按时间顺序写：

1. 你怎么开始用 OpenMy
2. 第一次跑通以后看到了什么
3. 哪个地方最让你觉得“终于不用自己记了”
4. 你后来怎么把它接给 Agent

## 结尾

只回答三件事：

1. 现在它替你省掉了什么
2. 你最常用哪三个动作
3. 你会推荐给哪种人

## 建议别写成什么样

- 别写成产品说明书
- 别写成空话
- 别一上来就讲架构

先讲你自己怎么用，别人自然就懂了。
