  <files>app/static/modules/playback.js</files>
  <read_first>
    - app/static/modules/playback.js（全文，重点看 getSegmentSceneTarget 函数中 duration 回退逻辑）
  </read_first>
  <action>
找到 playback.js 中 duration 硬编码 5 秒的回退逻辑（约 L44）：

```javascript
// 当前代码（伪精确）
const end = audioRef.offset_end || (start + 5);
```

改为使用 audio_ref 中的真实 duration：

```javascript
const realDuration = audioRef.duration_seconds || 0;
const end = audioRef.offset_end || (realDuration > 0 ? start + realDuration : start + 5);
```

这样：
- 有真实 duration → 播放完整 chunk
- 无 duration（老数据） → 回退到 5 秒（保持兼容）
  </action>

  </acceptance_criteria>
  <verify>node --check app/static/modules/playback.js && rg -n "duration_seconds\|realDuration" app/static/modules/playback.js</verify>
