# User story template

Use this template for the first public OpenMy story.

## Title

Say clearly what OpenMy saved you from.

Examples:
- How I turned a pile of recordings into working context with OpenMy
- How I stopped re-explaining my work to my agent every day

## Opening

Start with one real situation:

- why it felt painful
- what you did before
- why the old approach kept failing

## Middle

Tell it in time order:

1. how you started using OpenMy
2. what you saw the first time it worked
3. which moment made you think “I do not have to remember this manually anymore”
4. how you later connected it to your agent

## Ending

Answer only three questions:

1. what it saves you from now
2. which three actions you use most often
3. who you would recommend it to
